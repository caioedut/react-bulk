# Box

It is the primary component, factory for everything.

## Props

### `component`

Defines which element will be rendered.

| Type                   | Default (Web) | Default (Native) |
|------------------------|---------------|------------------|
| `string`, `ReactNode`  | `'div'`       | `View`           |

---

### `className`

For web only.

| Type     | Default |
|----------|---------|
| `string` | `null`  |
